# Legacy & Archived Themes

Welcome to the archives! This folder contains older, outdated, or deprecated versions of themes from this repository (such as V1 versions that have since been replaced by V2 upgrades).

### Important Notice
* **Not Actively Maintained:** These files are preserved exactly as they were. They will not receive regular bug fixes, layout repairs, or compatibility updates if Vencord, BetterDiscord or Discord changes.
* **Breakage Risk:** Because Discord updates frequently, elements in these older files may be visually broken or misaligned.

### Looking for the Best Version?
If you want the cleanest, most up-to-date look with the latest color updates and working assets, please use the main versions on the front page of the repository.
