# Vencord_Themes

> [!WARNING]
> Photosensitivity Notice: Some themes contain flashing GIF elements and highly vibrant backgrounds. Please use caution if you are sensitive to bright or fast-moving lights.

Custom Vencord themes inspired by popular anime. This collection features vibrant layouts blending high-quality anime wallpapers with dynamic, animated GIF backgrounds. Each theme includes meticulously matched color palettes for text, sidebars, and buttons to give your Discord client a personalized and cohesive aesthetic.

### Vencord Setup (Recommended)
* **Option A (Online Link):** Copy the Raw URL of the theme file, open Discord, go to User Settings > Vencord > Themes, paste the link into the "Online Themes" box, and press Enter.
* **Option B (File Download):** Download the `.css` file, open Discord, go to User Settings > Vencord > Themes, click "Open Themes Folder", and drop the file inside.

### BetterDiscord Setup
1. Download the `.css` file for the theme you want.
2. Open Discord and go to User Settings > Themes.
3. Click the "Open Themes Folder" button at the top.
4. Move the downloaded `.css` file into that folder and toggle it on.

## Device & Compatibility Notice
I made these themes specifically using client extensions on desktop. Because custom clients pull media differently across different platforms, the animated GIFs and wallpapers might not look right or load properly on mobile devices or certain client setups. For the best look, I highly recommend using them on the Vencord desktop app.
